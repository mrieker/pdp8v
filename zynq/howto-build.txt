

to build zynq zturn:

    ./makemodv.sh   ...create verilog equivalent of tube boards
    ./makentov.sh   ...adds nto counters (number of tubes on) to verilog

    build bitstream file with vivado2018.3 or try script ./makebitfile.sh
        ...creates file pdp8v.runs/impl_1/myboard_wrapper.bit

    create BOOT.BIN file with ./makebootbin.sh
        this requires ~/MYIR_HW contain directories
            petadevel   ...which contains petadevel/settings.sh
            petakernl   ...which contains petakernl/zturn/images/linux

to run it:
    cd pdp8v/driver
    ./raspictl -zynqlib -randmem -mintimes
